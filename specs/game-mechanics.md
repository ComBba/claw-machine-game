# 2D 인형뽑기 게임 메카닉 명세

## 목표
사이드뷰 2D 인형뽑기 게임 - 정확한 충돌 감지와 부드러운 애니메이션

## 요구사항

### 1. 집게 (Claw) 동작
- **대기**: 상단에서 좌우 이동 (◀ ▶ 버튼, 방향키)
- **하강**: GRAB 버튼 누르면 집게 전체가 아래로 내려감
  - ⚠️ 줄만 내려가면 안 됨! 집게 몸체도 함께!
- **잡기**: 바닥 도달 시 인형 감지 → 잡기
- **상승**: 잡은 인형과 함께 올라옴
  - ⚠️ 인형이 틀어지면 안 됨! 집게 중심에 고정
- **이동**: 배출구로 이동
- **배출**: 인형 떨어뜨림 → 점수

### 2. 충돌 감지 (2D 베스트 프랙티스)
- AABB (Axis-Aligned Bounding Box) 또는 원형 충돌
- 집게 중심점과 인형 중심점 간 거리 계산
- grabRadius 내에 있으면 잡기 성공

### 3. 좌표계 통일
- 모든 요소 픽셀(px) 기준
- 퍼센트(%) 사용 시 변환 함수 필수

### 4. 애니메이션
- requestAnimationFrame 사용
- 부드러운 이동 (선형 보간)
- 잡힌 인형은 집게 중심에 고정

## 검증 기준
- [ ] 집게 전체가 함께 내려감
- [ ] 인형이 집게 중심에 고정되어 올라옴
- [ ] 3회 연속 성공 가능
